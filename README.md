# Airbnb NYC Price Prediction using Machine Learning

## Overview

This project develops and evaluates machine learning models to predict Airbnb listing prices in New York City using real-world listing data.

The project follows a complete machine learning workflow including:

* Data cleaning and preprocessing
* Exploratory data analysis
* Feature engineering
* Model training and evaluation
* Hyperparameter tuning
* Model explainability using SHAP

## Dataset

The dataset contains approximately 48,000 Airbnb listings from New York City and includes features such as:

* Location
* Room type
* Number of reviews
* Availability
* Host information
* Pricing data

## Machine Learning Models

The following models were implemented and compared:

* Linear Regression
* Random Forest Regressor
* Gradient Boosting Regressor

Performance was evaluated using:

* R² Score
* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)

## Technologies Used

* Python

## Key Outcomes

* Developed an end-to-end machine learning pipeline.
* Improved predictive performance through hyperparameter optimisation.
* Applied explainable AI techniques using SHAP.
* Evaluated and compared multiple machine learning approaches.
